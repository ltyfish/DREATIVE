# Mobile

Use for every page.

Design 390px as its own composition: order, crop, type scale, spacing, density,
controls, sticky behavior, and motion may differ from desktop. Inspect 320px
when density or the audience makes narrow phones relevant.

- Keep readable content type and touch targets.
- Remove horizontal overflow and accidental viewport-width calculations.
- Replace persistent desktop ornament with an inline or bounded form when it
  competes with content.
- Keep touch, keyboard, reduced-motion, and core feature parity.
- Do not solve mobile by hiding the defining product content.

Inspect the full mobile route, not only the first viewport.
