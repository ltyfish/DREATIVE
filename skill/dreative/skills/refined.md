# Refined

Use when finish, typography, and material craft are central.

Define a clear type hierarchy, spacing rhythm, alignment system, palette, and
surface language. Make optical corrections where strict math looks wrong.
Balance dense passages with rest.

Reject generic shadows, arbitrary radii, excessive labels, hairline decoration
without structure, and tiny type used as a shortcut to sophistication.

Inspect real headings, long copy, cards, forms, and footer content at desktop
and mobile.
