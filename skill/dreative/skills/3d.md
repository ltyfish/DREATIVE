# 3D

Use 3D only when spatial behavior improves meaning, demonstration, or product
understanding.

Name the spatial contribution honestly: model, layered billboard, media plane,
procedural geometry, frame sequence, or pre-rendered view. A textured plane is
not a model.

Cap DPR, lazy load, suspend offscreen, handle resize/context failure, and
dispose every resource. Provide a lightweight fallback and an authored mobile
form.

Reject generic particles, floating decorative objects, invisible complexity,
and persistent canvases that cover copy or controls. Read
`../references/CREATIVE_EXECUTION.md` and `../recipes/3d-recipes.md`.
